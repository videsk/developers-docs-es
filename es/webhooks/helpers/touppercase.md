# toUpperCase

Este tipo de helper permite mutar un `String` convirtiendo todo texto a mayúsculas.

{% hint style="info" %}
Debes considerar que el valor a mutar debe ser un `String`, de lo contrario se retornará un error de formato.
{% endhint %}

#### Datos de ejemplo

```json
{ "name": "John Doe" }
```

## Ejemplos de uso

```handlebars
{{toLowerCase name}}
// Output: JOHN DOE
```
